# Gatling Spring Boot Application

This project integrates **Gatling** for load testing with a **Spring Boot** application written in **Kotlin**.

---


---

## Getting Started

### Build & Run

```bash
mvn clean install
```

or run directly:

```bash
mvn spring-boot:run
```

### Running Gatling

To execute Gatling scenarios:

```bash
mvn gatling:test
```
